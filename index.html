3.3 ISRC Registration Process

Every track released under TGR.Waves requires a unique International Standard Recording Code — the 'digital DNA' of a sound recording.

01

Apply to Indian Music Industry (IMI) for a Label Registrant Code — a unique 2-character prefix.

02

Once the prefix is assigned (e.g., 'IN-XYZ'), TGR.Waves can generate unlimited ISRCs internally.

03

Format: CC-XXX-YY-NNNNN (Country-Registrant-Year-Track Number).

04

Embed ISRC in metadata of every master file before delivery to distributors.

05

ISRC is a one-time registration per label. No renewal required.
